OSLiC @ Github

https://github.com/dtag-dbu/oslic

Das Github Repository (permanent | regelmäßig mitarbeiten)

  • Sie können das OSLiC Repository mit den Methoden holen, die über das OSLiC Github Frontend angeboten werden, insbesondere mit
    • git clone https://github.com/dtag-dbu/oslic.git
    • git clone git@github.com:dtag-dbu/oslic.git

Das OSLiC Repository ( regelmäßig mitarbeiten)

  • Natürlich können Sie das OSLiC Repository auch einfach im Sinne von Github forken. Die pull requests werden zu den wichtigsten Momenten versendet.

Die OSLiC Dateien ( fallweise mitarbeiten)

  • Sie können die OSLiC Dateien als Zip-Datei über den Zip-Button auf dem OSLiC Github frontend oder direkt über die folgende Url herunterlade:
    • https://github.com/dtag-dbu/oslic/zipball/master
