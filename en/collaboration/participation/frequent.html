Frequent Participation

If you are familiar with the use of GitHub and if you want to contribute regularly to the OSLiC development without being deeply embedded in the development team, you might fork the current master branch of the OSLiC Github repository for doing your own writer work. Finally you can mail the link of your own repository to opensource@telekom.de. We will merge your changings back into the OSLiC repository and send the corresponding pull requests.
